# 百度糯米(nuomi)

- 多年以前模仿百度糯米的首页网页代码特效展示 (jquery 版本)

### 克隆(clone)

- $ `git clone -b jquery git@github.com:johnnynode/nuomi.git  --depth 1`

### 安装(install)

- $ `yarn install` or $ `npm i`

### 预览（preview)

- $ `gulp server`

### 构建(build)

- $ `gulp build`

### 构建后的生产模式(build)

- $ `gulp build-server`
